Browser adapter

This is the easiest adapter to use, it can be used to target any browser that can run jQuery. The dependency on jQuery will be removed in the future.

It features implementations of many UI Elements optimized for the web.

It can be used to generate either "single-page" webapps for fully-fledged websites like the one you're currently browsing!

The browser adapter is also used as a base for other adapters in web-friendly devices, like the GoogleTV adapter.
